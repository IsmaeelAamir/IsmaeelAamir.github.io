Bootcamp 2020: Project

My project is a fun little game where you use a small blue ball to pop the green balls, whilst avoiding the red balls. Once all of the green balls have been popped, you win and can play another round. The size, position and ratio of red to green balls is random. This project makes particular use of arrays, loops and randomness. An array is used to store data about each ball, and a loop is used to populate the array, with random values for the size, positon and colour. The ball bounces off of the walls, and also bounces off the red balls. But if you hit a red ball, you lose a life, out of five lives. When the ball bounces off these objects, or pops a green ball, a noise is played. The mouse is used to aim, and the spacebar is used to shoot.

Java Game project

My project is a co-op 2 player game. It was originally based off of the game "Fireboy and Watergirl". I started to begin to make a clone of the game, but realised I would be unable to so for this project. Instead of restarting, I continued to use the sprites and made the game that I have here. When the water character touches fire, they lose a life and when the fire character touches water, they lose a life. lives can be regained through hearts. When all lives are lost, the player restarts
